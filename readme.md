### 🗺️ Lab Overview

In this challenge lab you will build a full end-to-end serverless solution for **Cymbal's Retro Cinema archive**:

| # | Task | Service Used |
|---|------|-------------|
| 1 | Create a Firestore database (Native mode) | Cloud Firestore |
| 2 | Import the Netflix CSV dataset | Firestore + Node.js |
| 3 | Deploy REST API v0.1 (baseline) | Cloud Run + Artifact Registry |
| 4 | Deploy REST API v0.2 (Firestore integration) | Cloud Run + Artifact Registry |
| 5 | Deploy the staging frontend | Cloud Run |
| 6 | Deploy the production frontend with live API | Cloud Run |

### 🔑 Key Concepts Covered

- **Cloud Firestore** — NoSQL document database in Native mode
- **Cloud Run** — Fully managed serverless container platform
- **Artifact Registry** — Docker image repository for container builds
- **Cloud Build** — CI/CD pipeline for building and pushing images
- **REST API Design** — Year-based filtering of the Netflix dataset
